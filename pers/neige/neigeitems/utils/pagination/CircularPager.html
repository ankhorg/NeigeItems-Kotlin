      • fromImmutableList

         static <T> CircularPager<T> fromImmutableList(@NotNull() List<T> handle, int pageSize, boolean clone)

        根据一个不可变 List 创建循环分页工具. 元素顺序为迭代器遍历顺序. 如果 clone 项为 true, 此后对该 List 的任何修改将无法反应到已创建的循环分页工具中. 如果 clone 项为 false, 此后对该 List 的任何修改将导致不可预知的错误. pageSize 小于等于 0 时将自动变为 1.

        Parameters:
        handle - 待分页 List
        pageSize - 页大小
        clone - 是否对传入的 List 进行复制
        Returns:

        分页工具
